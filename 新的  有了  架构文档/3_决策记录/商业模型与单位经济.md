## 一、ICP(理想客户画像)

### 1.1 第一目标:中国短视频创作者

**人群规模**:

- 抖音月活创作者 > 1 亿,持续输出(月发 ≥ 4 条)创作者 ≈ 2000 万
- 视频号 > 5000 万创作者
- 小红书内容创作者 ≈ 1500 万

**画像**:

- 年龄:22-40
- 角色:个人创作者 / 副业 / 自媒体 / KOC
- 月内容产出:**4-20 条短视频**
- 单条制作时间(自己做):2-6 小时(脚本 + 找素材 + 剪辑 + 配音)
- 平均月成本(自己时间 ¥80/小时):¥2000-9000
- 当前替代品:剪映会员(¥30/月)+ 自己

**痛点优先级**:

1. **时间**:每周 2-5 条,占 8-30 小时
2. **创意瓶颈**:脚本卡壳、选题难
3. **质量不稳**:有的爆有的没人看
4. **配音/字幕**:口播费时,字幕对齐烦
5. **多平台适配**:同一内容要发 3 个平台(竖屏 / 横屏)

### 1.2 第二目标:中小电商商家(详情图)

**人群规模**:

- 淘宝 / 拼多多 / 抖音电商 中小商家 ≈ 1500 万
- 每月新品上架次数:1-50

**画像**:

- 角色:个人卖家 / 小型电商团队 营销岗
- 痛点:每个新品要 1 张精美详情图,设计师 ¥200-1000/张,模板套来套去没特色
- 当前替代品:Canva / 稿定 / 即梦

### 1.3 V2+ 拓展(不在 V1 范围)

- 教育内容创作者(知识科普 / 课程营销)
- 本地生活商家(餐饮 / 美容 / 教培)
- 出海创作者(英文 TikTok / Instagram / YouTube Shorts)

## 二、定价(V1 上线即生效)

### 2.1 套餐设计

| 套餐 | 月价 | 视频任务/月 | 图片任务/月 | 长任务并发 | 适合 |
|-----|------|-----------|------------|----------|-----|
| **免费版** | ¥0 | 5 | 30 | 1 | 试用 / 极轻度 |
| **创作者版** | **¥69** | 30 | 不限 | 2 | 个人创作者 |
| **专业版** | **¥199** | 100 | 不限 | 4 | SMB / KOL |
| **团队版** | ¥599 | 500 | 不限 | 10 | 工作室 |
| **企业版** | 议价 | 不限 | 不限 | 不限 | 大型 MCN / 品牌 |

**附加**:

- 超量包:¥4/视频(创作者版)/ ¥3/视频(专业版)
- 年付 8 折(¥69 → ¥663/年,等同 9 折)

### 2.2 Benchmark(竞品定价)

| 产品 | 月价 | 备注 |
|-----|------|------|
| Synthesia | $30 / $90 / $300 | 数字人视频,海外 |
| HeyGen | $24 / $72 | 视频化身 |
| Runway | $15 / $35 / $95 | 视频生成 |
| Pika | $10 / $35 / $95 | 视频生成 |
| 剪映会员 | ¥30 | 编辑工具,非生成 |
| 即梦会员 | ¥39-99 | 字节图像生成 |
| Canva | ¥49 | 通用设计 |
| 稿定设计 | ¥39 | 设计工具 |

「有了」定价**比国内编辑工具(¥30-50)贵 1-3 倍,比国外生成工具(¥150-300 等价)便宜 50%**——锁定"中等价位、高度自动化"区间。

### 2.3 阶梯计费(关键)

视频任务**模型成本高,固定额度可能亏损**(详见 §3),因此用阶梯计费:

```
创作者版 ¥69:
  - 包含 30 条视频额度
  - 超量 ¥4/条
  - 实际平均使用 15 条 → 不超额
  - 重度用户(月 50 条)→ ¥69 + 20 × ¥4 = ¥149(仍便宜过专业版的全包)

专业版 ¥199:
  - 包含 100 条额度
  - 超量 ¥3/条
  - 实际平均使用 50 条 → 不超额

边界设计:让 50% 用户不超量,30% 微超(每月 +¥20-50),10% 重超(自然升级)
```

## 三、单位经济测算(透明账)

### 3.1 单条 1 分钟短视频成本(模型 + 工具,v3.0 编号)

| 项 | 调用 | 成本 |
|----|------|------|
| 主编排 LLM(deepseek-v4-flash 15 次)| 15 × $0.0002 | $0.003 |
| Agent 1 脚本生成(kimi-k2 5K tokens 输入 + 1K 输出)| | $0.05 |
| **Agent 3 图像生成**(seedream-3 × 5)| | $0.30 |
| **Agent 4 TTS**(volcengine 60s)| 60s × $0.001 | $0.05 |
| **Agent 4 视频合成**(image-to-video × 5 个 12s 片段)| 5 × $0.10 | **$0.50** |
| 字幕(faster-whisper 自托管,GPU 摊销)| | $0.005 |
| OSS 存储 + 流量 | | $0.02 |
| **小计模型成本** | | **$0.93** |
| Agent 实例 / Celery 时间(GPU 节点摊销)| | $0.20 |
| **总成本(中位数)** | | **$1.13 ≈ ¥8** |

**优化点**:

- 5 张图 → 4 张(减 $0.06)
- image-to-video 用更短片段(8s × 6 替代 12s × 5)→ 视频部分 $0.40
- 重度用户用更便宜的 seedance-1(quality 略低)→ 视频部分 $0.30
- **可压到 ¥6/条**

**最坏情况**(全部走最贵模型,Veo-3):¥15/条

### 3.2 单条详情图成本

| 项 | 成本 |
|----|------|
| 主编排 LLM | $0.003 |
| Agent 1 文案 5 段 | $0.02 |
| Agent 2 风格分析 | $0.005 |
| Agent 2 batch_generate × 5 | $0.30 |
| Agent 4 长图拼接 | $0(纯 PIL)|
| Agent 2 质检 | $0.005 |
| OSS | $0.005 |
| **总成本** | **$0.34 ≈ ¥2.5** |

详情图成本低,定价相对宽松,主要是覆盖 image generation 成本。

### 3.3 毛利分析(创作者版 ¥69)

假设 3 类用户分布:

| 用户类型 | 占比 | 月用视频条数 | 模型成本 | 用户支付 | 毛利 |
|---------|------|------------|---------|---------|------|
| 轻度 | 40% | 5 | ¥40 | ¥69 | ¥29 |
| 中度 | 40% | 15 | ¥120 | ¥69 | -¥51 |
| 重度 | 15% | 30(刚到额度) | ¥240 | ¥69 | -¥171 |
| 超量 | 5% | 50(超 20) | ¥400 | ¥69+¥80=¥149 | -¥251 |

**加权平均毛利**:

```
0.4 × 29 + 0.4 × (-51) + 0.15 × (-171) + 0.05 × (-251)
= 11.6 - 20.4 - 25.65 - 12.55
= -¥47/用户/月
```

**结论**:创作者版**当前模型成本下亏损 ¥47/月/用户**。

### 3.4 毛利分析(专业版 ¥199)

| 用户类型 | 占比 | 月用视频条数 | 模型成本 | 用户支付 | 毛利 |
|---------|------|------------|---------|---------|------|
| 轻度 | 25% | 20 | ¥160 | ¥199 | ¥39 |
| 中度 | 50% | 50 | ¥400 | ¥199 | -¥201 |
| 重度 | 20% | 100 | ¥800 | ¥199 | -¥601 |
| 超量 | 5% | 150(超 50) | ¥1200 | ¥199+¥150=¥349 | -¥851 |

**专业版亏损更狠** —— 因为重度用户多,亏穿。

### 3.5 怎么活?

3 个杠杆:

#### 杠杆 1:压模型成本(短期可控)

- image-to-video 用更短片段、更便宜模型 → 视频成本压到 ¥3/条
- 部分场景纯图 + Ken Burns 效果(无 AI 视频生成)→ 成本 ¥1.5/条
- BGM 用平台素材库(已购买商用授权,边际成本 0)
- TTS 自托管(V2)→ 成本 ¥0.1/条

**重新测算(优化后,创作者版)**:

```
轻度 5 条:¥15 成本,¥69 收入,毛利 ¥54
中度 15 条:¥45 成本,¥69 收入,毛利 ¥24  
重度 30 条:¥90 成本,¥69 收入,毛利 -¥21
超量 50 条:¥150 + 超量收 ¥80,合计 ¥149,毛利 -¥1
加权:0.4×54 + 0.4×24 + 0.15×(-21) + 0.05×(-1) = ¥27/用户/月
```

→ **优化后创作者版毛利 ¥27/月,可持续**。

#### 杠杆 2:用补贴换增长(中期)

前 6 个月愿意亏损,换月活 100K 量级。融资 pitch:

> "短视频赛道每用户月 LTV ¥300+,前 6 个月每用户补贴 ¥50,完全可接受。当前烧钱速率 ¥500K/月,12 个月内毛利转正。"

#### 杠杆 3:自研模型(长期)

V2 训练自有视频模型(基于开源 + 用户数据微调):

- 单条视频成本压到 ¥1-2
- 毛利率 80%+
- 数据飞轮(ADR-011)持续优化

## 四、CAC / LTV 假设

### 4.1 CAC 渠道分布

| 渠道 | CAC | 占比(目标)|
|-----|-----|----------|
| 抖音 KOL 投放 | ¥50-80 | 30% |
| 小红书内容种草 | ¥30-50 | 25% |
| 视频号自然流量 | ¥10-20 | 15% |
| 知乎 / 即刻 / 公众号 | ¥40-60 | 10% |
| 用户邀请奖励 | ¥20(等价优惠券)| 15% |
| 自然搜索 / SEO | ¥5-10 | 5% |

**加权平均 CAC**:¥42

### 4.2 LTV 假设

```
ARPU 测算:
- 免费版 50%:¥0
- 创作者版 30%:¥69
- 专业版 18%:¥199
- 团队版 2%:¥599

加权 ARPU = 0.5 × 0 + 0.3 × 69 + 0.18 × 199 + 0.02 × 599
         = 0 + 20.7 + 35.8 + 12 
         = ¥68.5/用户/月

月留存目标:
- 第 1 月 → 第 2 月:60%
- 第 2 月 → 第 3 月:75%
- 长期月留存:85%

平均生命周期 = 1 / (1 - 0.85) = 6.7 个月(粗算)

LTV = ¥68.5 × 6.7 ≈ ¥459
```

### 4.3 LTV / CAC

¥459 / ¥42 = **10.9** ✓(健康标准 > 3)

如果 CAC 控制不住涨到 ¥80,LTV/CAC = 5.7,仍健康。

## 五、GTM(go-to-market)

### 5.1 Phase 1(0-3 月,封测)

**目标**:50-200 个种子用户,验证 PMF。

**渠道**:

- 创始团队朋友圈
- 内测申请(发"内测申请"链接到知乎 / 即刻 / 微信群)
- 5 个 10-50 万粉短视频创作者(种子 KOC)

**关键指标**:

- 周留存 > 30%
- 单用户月使用 > 5 次
- NPS > 30

**预算**:¥10 万(种子 KOC 内容置换 + 服务器成本)

### 5.2 Phase 2(3-6 月,公测)

**目标**:DAU 5K,月活 30K,付费用户 1K(转化率 3%)

**渠道**:

- 抖音 KOL 投放(10 个 50-200 万粉,¥10 万预算)
- 小红书 KOC 投放(30 个 1-10 万粉,¥3 万)
- 视频号自然流量(产品分享群名片)
- 知乎 / 即刻 / 公众号内容运营(团队 1 个内容岗)
- 用户邀请激励(分享群名片返优惠券)

**关键指标**:

- CAC < ¥60
- 周留存 > 40%
- 付费转化率 > 3%

**预算**:¥30 万投放 + ¥20 万内容运营人力 = ¥50 万

### 5.3 Phase 3(6-12 月,扩张)

**目标**:DAU 50K,月活 200K,付费用户 8K,**MRR ¥500K**(MRR = 月经常性收入)

**新动作**:

- **创作者飞轮 V1.5 上线**(Skill 市场内测,50 创作者种子)
- **B 端商家拓展**(淘宝 / 小红书 商家)
- **海外测试**(TikTok 创作者,英文版 V2)

**预算**:¥200-300 万(投放 + 创作者激励 + 团队扩张)

### 5.4 Phase 4(12+ 月,规模化)

**目标**:MRR ¥3M+, 准备 A 轮 / B 轮融资

**节点**:

- V2 自研模型上线 → 毛利率 70%+
- 团队版 / 企业版客户 50+
- 海外市场启动

## 七、关键风险与应对

### 7.1 模型成本失控

**风险**:foundation model 提价(GPT-5 / Claude / Veo 等),单条视频成本翻倍。

**应对**:
- LiteLLM L2 路由灵活切换更便宜的备用
- 自托管 BGE-M3 / faster-whisper / 部分图像模型
- V2 自研短视频模型(专门优化短视频场景,不需要通用能力)

### 7.2 ICP 错位

**风险**:短视频创作者实际不愿意付 ¥69/月。

**应对**:
- 封测期密切监控 NPS 和付费转化
- A/B 测试不同价格点(¥39 / ¥69 / ¥99)
- 如果发现 SMB 营销岗(B 端)付费意愿更强,转向 B 端

### 7.3 同期对手收割

**风险**:Coze / 字节 / 阿里 推出类似产品。

**应对**:
- **数据飞轮**(ADR-011)是核心防御
- **Skill 市场**(V1.5)建立创作者生态
- **垂直深度**(短视频细到平台优化)而非广度

### 7.4 监管

**风险**:生成式 AI 备案 / 内容审核违规。

**应对**:
- 先做生成式 AI 服务备案
- 所有用户上传 + 模型输出 过阿里云内容安全
- 政治敏感内容硬过滤

### 7.5 失败模式(假设最坏情况)

如果 12 个月内:

- DAU < 5K
- 月留存 < 30%
- 付费转化 < 1%

→ **承认 PMF 不在,pivot**:

- Plan B:转 B 端 SaaS(MCN 公司 / 电商商家工具)
- Plan C:开放 API,做底层 agent 平台(AI infra)
- Plan D:卖掉团队 / acqui-hire
